# Koinsave Frontend Project

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Short Overview
A simple frontend app for Koinsave with login, signup, dashboard, and mock transactions using Next.js and MockAPI.

## Technologies
Next.js, React, Tailwind CSS, Axios, MockAPI

## Setup Instructions

1. Clone the repo
   ```bash
   git clone https://github.com/vickywagz/koinsave-frontend.git

2. Install dependencies

    npm install

3. Run development server

    npm run dev


4. Open in browser

    http://localhost:3000
